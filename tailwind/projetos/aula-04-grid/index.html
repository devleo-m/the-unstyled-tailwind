Aula 4 · CSS Grid com Tailwind CSS

Agora que você já domina Flexbox, é hora de coordenar linhas e colunas com o Grid. Cada classe do Tailwind corresponde a uma propriedade CSS clara e fácil de memorizar.

grid gap col-span auto-fit

Como professor, eu quero que você entenda o “porquê” do layout: cada seção abaixo desconstrói um pedaço da aula com explicações, analogias e exemplos reais.

Fundamentos

Grid, colunas e linhas

O Tailwind mapeia diretamente o que você já conhece em CSS: `grid` equivale a `display: grid`, e cada `grid-cols-*` ou `grid-rows-*` define a estrutura base do layout.

Grid container

Use `grid` para ativar o modo bidimensional. Os filhos viram células que respeitam colunas e linhas.

<div class="grid">
<div>A1</div>
<div>A2</div>
</div>

CSS: `display: grid;`

Colunas

`grid-cols-3` divide o grid em três colunas iguais. Pense na repetição `repeat(3, minmax(0, 1fr))` em CSS.

<div class="grid grid-cols-3 gap-4">
<div>1</div>
<div>2</div>
<div>3</div>
</div>

CSS: `grid-template-columns: repeat(3, minmax(0, 1fr));`

Linhas

`grid-rows-3` força três linhas; útil quando você quer controlar o tamanho vertical.

<div class="grid grid-rows-3 gap-4">
<div>A</div>
<div>B</div>
<div>C</div>
</div>

CSS: `grid-template-rows: repeat(3, minmax(0, 1fr));`

Gap & Spanning

Espaçamento e células maiores

`gap` respira entre as células, `col-span` e `row-span` esticam elementos para ocupar mais de uma coluna ou linha. Pense em blocos de notas juntando páginas.

Gap

`gap-6` coloca 1.5rem entre colunas e linhas ao mesmo tempo.

<div class="grid grid-cols-3 gap-6">
<div>Card</div>
<div>Card</div>
<div>Card</div>
</div>

CSS: `gap: 1.5rem;`

Column span

`col-span-2` faz um cartão largo, ideal para destaque.

<div class="grid grid-cols-4 gap-4">
<div class="col-span-2">Feature</div>
<div>Item</div>
</div>

CSS: `grid-column: span 2 / span 2;`

Template Areas

Mapeando áreas nomeadas

Ainda que o Tailwind não tenha utilitário direto, você pode complementar com `style` + `@apply` para criar áreas nomeadas.

<div class="grid grid-cols-4 gap-4" style="grid-template-areas: 'header header header header' 'sidebar main main aside' 'footer footer footer footer';">
<header style="grid-area: header;">Header</header>
<aside style="grid-area: sidebar;">Sidebar</aside>
<main style="grid-area: main;">Conteúdo</main>
<aside style="grid-area: aside;">Destaques</aside>
<footer style="grid-area: footer;">Footer</footer>
</div>

Use `@apply` para mover esse CSS para um arquivo e manter o HTML limpo.

Auto-fit / Auto-fill

Galerias fluidas

Com `repeat(auto-fit, minmax())` você permite que o grid cresça e encolha com o tamanho da tela sem quebrar o layout.

<div class="grid gap-4" style="grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));">
<div class="rounded-xl bg-slate-100 p-4">Card</div>
…
</div>

CSS final: `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));`

Auto-fit distribui o máximo de colunas possíveis.
Auto-fill respeita o tamanho mínimo e deixa “espaços vazios”.
Combine com `minmax()` para definir limites de largura.

Responsividade

Prefixos mobile-first

Comece com `grid-cols-1` e depois aumente com `md:grid-cols-2`, `lg:grid-cols-3`, etc.

<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">

Adicione `md:col-span-2` para esticar elementos em telas maiores (por exemplo, destaque em desktop).

Mobile: 1 coluna e layout vertical.
Desktop: 3 colunas com cards lado a lado.

  • Grid com `grid-cols-12`.
  • Sidebar `lg:col-span-3`, conteúdo `lg:col-span-9`.
  • `gap-6` para respirabilidade e `lg:gap-8` no desktop.

Exercício 2 · Cartões responsivos

  • Use `repeat(auto-fit, minmax(220px, 1fr))`.
  • Combine `gap-x-6` e `gap-y-4` para eixos diferentes.
  • Destaque o card principal com `col-span` em `lg:`.

Conexão com CSS puro

Cada classe conversa com o que você já conhece

Grid container

`grid` = `display: grid`

`grid-cols-3` = `grid-template-columns: repeat(3, minmax(0, 1fr))`

Spanning

`col-span-2` = `grid-column: span 2 / span 2`

`row-span-3` = `grid-row: span 3 / span 3`

Gap e alinhamento

`gap-6` = `gap: 1.5rem`

`place-items-center` = `place-items: center`
